## Setup
- Clone the repo
- copy the `.env.sample` into an `.env` file:
```shell
cp .env.sample .env
```
- replace `NASA_API_KEY` with an actual NASA API KEY created [here](https://api.nasa.gov/#browseAPI)
- install Node packages: `yarn install`
- start dev server: `yarn dev`
